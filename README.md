# mysq-express
Building REST APIs with MySQL and Node.js 

Installation:
=============

1-clone prokect and install npm
$sudo npm install 

2-config database:

3-create new  database in your local server and change the configuration 

in :/config/config.js

module.exports = {
     hostname:'localhost',
     username:'root',
     password:'password',
     database:'data_base_name',
     user_email:'miranavvs@gmail.com',
     user_password :'toto'
}

3-start projetc :

$cd /path_projetc

$ npm start

congratulation....!
